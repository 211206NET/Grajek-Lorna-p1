## Functionality

* Ability to create new stores, manage existing stores, and update their product inventory
* User sign up and log in with input validation
* Purchase multiple products in a single order
* View order history by user or store
* Database connection using SQL Server and AWS
* Implements logging for each significant event in the application
* Implements teseting using Xunit

## Get Started

To checkout the store application use the following steps:

> Download dotnet SDK or dotnet Run Time, and Visual Studio <br>
> Clone the repo `git clone https://github.com/211206NET/Grajek-Lorna-p1.git` <br>
> Navigate to WebAPI folder and run the program using Vistual Studio or CLI
